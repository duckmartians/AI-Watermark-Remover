## ইনস্টল ও চালু করা

1. ইনস্টলারটি চালান (অথবা যদি আপনি একটি সংকুচিত আর্কাইভ পেয়ে থাকেন তবে ফোল্ডারটি আনজিপ করুন)।
2. **AI Watermark Remover** খুলুন (ডেস্কটপ, স্টার্ট মেনু, অথবা `.exe` ফাইল থেকে)।

> **Windows কি "Windows protected your PC" (Windows আপনার পিসি রক্ষা করেছে) দেখাচ্ছে?**
> এটি হলো পেইড সাইনিং সার্টিফিকেট ছাড়া অ্যাপগুলোর জন্য ডিফল্ট সতর্কবার্তা — **এটি কোনো ভাইরাস নয়**।
> এটি খুলতে **More info → Run anyway** (আরও তথ্য → যেভাবেই হোক চালান) ক্লিক করুন।

প্রথমবার চালু হতে একটু বেশি সময় লাগতে পারে; পরবর্তী চালুগুলো দ্রুত হয়।

**দুটি সংস্করণ আছে:** **Pro** সংস্করণ (ছবি + ভিডিও) এবং **Lite** সংস্করণ (শুধু ছবি, হালকা)।
এগুলো আলাদা করতে উইন্ডোর শিরোনাম দেখুন: **"Pro"** = পূর্ণ সংস্করণ, **"Lite"** = হালকা সংস্করণ (ভিডিও খুলতে পারে না)।

## সিস্টেম প্রয়োজনীয়তা

| | **Lite** (ছবি) | **Pro** (ছবি + ভিডিও) |
|---|---|---|
| প্ল্যাটফর্ম | **Windows 10/11** 64-bit **এবং macOS** | **শুধুমাত্র Windows 10/11 64-bit** — *macOS নেই* |
| CPU | **AVX2** সহ 64-bit | **AVX2** সহ 64-bit |
| GPU | **প্রয়োজন নেই** — CPU-তে চলে | **CUDA সহ NVIDIA GPU আবশ্যক** |
| VRAM | — | ন্যূনতম 4 GB · **6–8 GB প্রস্তাবিত** (HD / দীর্ঘ ক্লিপ) |
| RAM | 4 GB+ | 8 GB+ |
| আরও ইনস্টল করুন (Windows) | **Visual C++ Redistributable 2015–2022 x64** | একই + একটি হালনাগাদ **NVIDIA driver** |

- **Pro** সংস্করণ তার ভিডিও AI চালায় **NVIDIA CUDA কোর**-এ, তাই এটি **শুধুমাত্র Windows-এর জন্য এবং এর জন্য একটি NVIDIA গ্রাফিক্স কার্ড প্রয়োজন** — মোটামুটি **RTX 2060 (RTX 20-series / GTX 16-series, "Turing") বা নতুনতর**।
- **macOS-এ CUDA নেই** → Mac-এ **Lite** সংস্করণ (ছবি) ব্যবহার করুন। Pro-এর ভিডিও ইঞ্জিনে AMD/Intel GPU-ও **সমর্থিত নয়**।
- **ছবি** থেকে সরানোর কাজ **দুটি** সংস্করণেই **CPU-তে** ভালোভাবে চলে — কোনো গ্রাফিক্স কার্ডের দরকার নেই।

## কীভাবে ব্যবহার করবেন — ৪টি ধাপ

### ধাপ ১ — একটি ছবি/ভিডিও খুলুন
- এক বা একাধিক ফাইল বাছাই করতে **ডকুমেন্ট** বোতামে ক্লিক করুন, অথবা পুরো ফোল্ডার খুলতে **ফোল্ডার** বোতামে ক্লিক করুন।
- অথবা ছবি/ভিডিও সরাসরি উইন্ডোতে **টেনে এনে ছেড়ে দিন**।
- একাধিক ফাইল থাকলে নিচে একটি থাম্বনেইল স্ট্রিপ পাবেন; প্রতিটি আইটেমে একটি **×** আছে যা তালিকা থেকে সেটি সরিয়ে দেয়।

### ধাপ ২ — যা সরাতে চান তার উপর রং করুন
বাম কলামে একটি টুল বাছাই করুন, তারপর ওয়াটারমার্কের উপর রং/রূপরেখা আঁকুন (রং করা জায়গাটি **নরম লাল** রঙে দেখায়):

| টুল | কী কাজে লাগে |
|----|---|
| **Brush** | ওয়াটারমার্কের উপর রং করুন (বাম মাউস বোতাম চেপে ধরে টানুন)। নিচের স্লাইডার দিয়ে ব্রাশের আকার সমন্বয় করুন। |
| **Box** | দ্রুত ঢেকে দিতে ওয়াটারমার্কের উপর একটি আয়তক্ষেত্র টানুন। |
| **Eraser** | বাড়তি রং মুছে ফেলুন। |
| **Text** | একটি লেখা-ধরনের ওয়াটারমার্ক নিখুঁতভাবে ঢাকতে লেখা টাইপ করুন। এই লেখাটি শুধু একটি **চিহ্ন** — প্রসেসিংয়ের পরে এটি মিলিয়ে যায় এবং ছবিতে স্থায়ীভাবে থেকে যায় না। |

খুব বেশি রং করে ফেলেছেন? বাম কলামে **Undo / Redo / Clear all / Reset** ব্যবহার করুন।
টুলটি বাতিল করতে আবার টুল বোতামে ক্লিক করুন (অথবা **Esc** চাপুন)।

> **টিপ:** ওয়াটারমার্ক পুরোপুরি ঢাকুন এবং একটু বাইরে ছড়িয়ে দিন — কম রং করার চেয়ে ফলাফল আরও ভালো দেখায়।

### ধাপ ৩ — Run ক্লিক করুন
- **Run** — বর্তমানে খোলা ছবি/ভিডিওটি প্রসেস করুন।
- **Run all** — তালিকার **একই ফ্রেম আকারের** প্রতিটি ফাইল ব্যাচ-প্রসেস করুন
  (শুধু ২টি বা তার বেশি ফাইল থাকলে সক্রিয় হয়)।

### ধাপ ৪ — সংরক্ষণ করুন
- **Save** ক্লিক করুন। **প্রতিবার সংরক্ষণ করার সময় অ্যাপটি আপনাকে আউটপুট ফাইলের জন্য একটি ফোল্ডার বেছে নিতে বলে**।
- মূল ফাইলগুলো রাখতে চান? **"Add _clean"** অপশনটি সক্রিয় করুন — নতুন ফাইলের নামে একটি `_clean` প্রত্যয়
  যুক্ত হয় (যেমন `photo.png` → `photo_clean.png`), যাতে মূল ফাইলটি ওভাররাইট না হয়।

## ভিডিও প্রসেসিং (Pro সংস্করণ)

- ভিডিও **Save বোতাম দিয়ে সংরক্ষণ করা হয় না**। **Run** (একটি ভিডিও) অথবা **Run all**
  (একাধিক ভিডিও) ক্লিক করুন; শুরু হওয়ার সময় অ্যাপটি একটি সংরক্ষণ ফোল্ডার চায়।
- ভিডিও AI দিয়ে ফ্রেম-বাই-ফ্রেম প্রসেস করা হয়, তাই এটি **ছবির চেয়ে অনেক ধীর** — শুধু এটিকে চলতে দিন।

### গতি ⇄ মান স্লাইডার

**ব্রাশের আকার** স্লাইডারের নিচে (বাম কলাম, শুধু **Pro**) গতি ও মানের মধ্যে ভারসাম্য করার জন্য চারটি স্লাইডার আছে। যেকোনো স্লাইডারের উপর মাউস রাখলে একটি টিপ দেখা যায়; আপনার সেটিংস মনে রাখা হয়।

| স্লাইডার | এটি কী নিয়ন্ত্রণ করে | দ্রুততর | তীক্ষ্ণতর |
|---|---|---|---|
| **Resolution** | প্রসেসিং রেজোলিউশন — সবচেয়ে বড় গতি নিয়ন্ত্রক। কম = অনেক দ্রুত, মোছা জায়গা একটু নরম। | ~**50** | **100** |
| **Motion** | ফ্রেমগুলোর মধ্যেকার গতি কতটা পুঙ্খানুপুঙ্খভাবে বিশ্লেষণ করা হয় (মান খুব একটা বদলায় না)। | **2–4** | 20 |
| **Frames** | কতগুলো কাছাকাছি ফ্রেম একসাথে সামলানো হয়। | কম | বেশি |
| **Reference** | রেফারেন্স ফ্রেম বাছাই করার সময়ের ব্যবধান। | **বেশি** | কম |

- দ্রুত চান? **Resolution ≈ 50** এবং **Motion ≈ 4** সেট করুন (ডিফল্ট মানগুলো এমনিতেই গতির পক্ষে)।
- মোছা জায়গা ঝাপসা দেখাচ্ছে? **Resolution** বাড়িয়ে **100**-এর দিকে নিন।
- এগুলো শুধু **ভিডিও**-তে প্রভাব ফেলে; ছবিতে কোনো প্রভাব পড়ে না।

## দেখা ও টুলস

- **মাউস হুইল**: জুম ইন / আউট · **মাঝের মাউস চেপে টানুন**: প্যান · **F** অথবা **ডাবল-ক্লিক**: উইন্ডোতে ফিট।
- **ঘর** আইকন: হোমপেজ খুলুন। **গ্লোব** আইকন: ভাষা পরিবর্তন করুন (৯টি ভাষা সমর্থিত, ভিয়েতনামিসহ)।

## কীবোর্ড শর্টকাট

| কাজ | কী |
|---|---|
| Undo / Redo | `Ctrl+Z` / `Ctrl+Shift+Z` |
| ব্রাশের আকার পরিবর্তন | `Ctrl` + মাউস হুইল |
| উইন্ডোতে ফিট | `F` অথবা ডাবল-ক্লিক |
| টুল বাতিল করুন | `Esc` |
| দ্রুত মোছা (Brush ব্যবহারের সময়) | **ডান মাউস বোতাম** চেপে ধরুন |

## इंस्टॉल करें और खोलें

1. इंस्टॉलर चलाएँ (या यदि आपको एक संपीड़ित संग्रह मिला है तो फ़ोल्डर को अनज़िप करें)।
2. **AI Watermark Remover** खोलें (डेस्कटॉप से, स्टार्ट मेन्यू से, या `.exe` फ़ाइल से)।

> **Windows में "Windows protected your PC" (Windows ने आपके पीसी को सुरक्षित किया) दिखता है?**
> यह उन ऐप्स के लिए डिफ़ॉल्ट चेतावनी है जिनके पास भुगतान किया गया साइनिंग प्रमाणपत्र नहीं होता — **यह वायरस नहीं है**।
> इसे खोलने के लिए **More info → Run anyway** (अधिक जानकारी → फिर भी चलाएँ) पर क्लिक करें।

पहली बार खोलने में थोड़ा अधिक समय लग सकता है; बाद के लॉन्च तेज़ होते हैं।

**दो संस्करण उपलब्ध हैं:** **Pro** संस्करण (छवियाँ + वीडियो) और **Lite** संस्करण (केवल छवियाँ, हल्का)।
इन्हें पहचानने के लिए विंडो के शीर्षक को देखें: **"Pro"** = पूर्ण संस्करण, **"Lite"** = हल्का संस्करण (वीडियो नहीं खोल सकता)।

## उपयोग कैसे करें — 4 चरण

### चरण 1 — कोई छवि/वीडियो खोलें
- एक या अधिक फ़ाइलें चुनने के लिए **दस्तावेज़** बटन पर क्लिक करें, या पूरा फ़ोल्डर खोलने के लिए **फ़ोल्डर** बटन पर क्लिक करें।
- या छवियों/वीडियो को सीधे विंडो में **खींचें और छोड़ें (drag and drop)**।
- कई फ़ाइलों के साथ आपको नीचे थंबनेल पट्टी मिलती है; प्रत्येक आइटम में उसे सूची से हटाने के लिए एक **×** होता है।

### चरण 2 — जो हटाना है उस पर पेंट करें
बाएँ स्तंभ में एक टूल चुनें, फिर वॉटरमार्क पर पेंट करें/उसकी रूपरेखा बनाएँ (पेंट किया गया क्षेत्र **हल्के लाल** रंग में दिखता है):

| टूल | इसका उपयोग किसलिए |
|----|---|
| **Brush** | वॉटरमार्क पर पेंट करें (बायाँ माउस बटन दबाकर रखें और खींचें)। नीचे दिए स्लाइडर से ब्रश का आकार समायोजित करें। |
| **Box** | त्वरित कवर के लिए वॉटरमार्क पर एक आयत खींचें। |
| **Eraser** | अतिरिक्त पेंटिंग को मिटा दें। |
| **Text** | टेक्स्ट-शैली के वॉटरमार्क को सटीक रूप से ढकने के लिए टेक्स्ट टाइप करें। यह टेक्स्ट केवल एक **मार्कर** है — प्रोसेसिंग के बाद यह गायब हो जाता है और छवि में शामिल नहीं होता। |

बहुत ज़्यादा पेंट कर दिया? बाएँ स्तंभ में **Undo / Redo / Clear all / Reset** का उपयोग करें।
टूल को अचयनित करने के लिए उसके बटन पर फिर से क्लिक करें (या **Esc** दबाएँ)।

> **सुझाव:** वॉटरमार्क को पूरी तरह ढकें और थोड़ा बाहर तक फैलाएँ — कम पेंट करने की तुलना में परिणाम बेहतर दिखते हैं।

### चरण 3 — Run पर क्लिक करें
- **Run** — वर्तमान में खुली छवि/वीडियो को प्रोसेस करता है।
- **Run all** — सूची में **समान फ़्रेम आकार वाली** हर फ़ाइल को बैच-प्रोसेस करता है
  (केवल तभी सक्षम होता है जब 2 या अधिक फ़ाइलें हों)।

### चरण 4 — सहेजें
- **Save** पर क्लिक करें। **हर बार सहेजते समय, ऐप आपसे आउटपुट फ़ाइलों के लिए एक फ़ोल्डर चुनने को कहता है**।
- मूल फ़ाइलें रखना चाहते हैं? **"Add _clean"** विकल्प सक्षम करें — नई फ़ाइलों के नाम में `_clean` प्रत्यय जुड़ जाता है
  (उदाहरण: `photo.png` → `photo_clean.png`), जिससे मूल फ़ाइल पर कुछ नहीं लिखा जाता।

## वीडियो प्रोसेसिंग (Pro संस्करण)

- वीडियो **Save बटन से नहीं सहेजे जाते**। **Run** (एक वीडियो) या **Run all**
  (कई वीडियो) पर क्लिक करें; शुरू होते ही ऐप सहेजने के लिए एक फ़ोल्डर माँगता है।
- वीडियो को AI द्वारा फ़्रेम दर फ़्रेम प्रोसेस किया जाता है, इसलिए यह **छवियों की तुलना में बहुत धीमा** होता है — बस इसे चलने दें।

## देखना और उपयोगिताएँ

- **माउस व्हील**: ज़ूम इन / आउट · **मध्य माउस बटन दबाकर खींचें**: पैन करें · **F** या **डबल-क्लिक**: विंडो में फ़िट करें।
- **घर** आइकन: होमपेज खोलें। **ग्लोब** आइकन: भाषा बदलें (वियतनामी सहित 9 भाषाएँ समर्थित)।

## कीबोर्ड शॉर्टकट

| क्रिया | कुंजी |
|---|---|
| Undo / Redo | `Ctrl+Z` / `Ctrl+Shift+Z` |
| ब्रश का आकार बदलें | `Ctrl` + माउस व्हील |
| विंडो में फ़िट करें | `F` या डबल-क्लिक |
| टूल अचयनित करें | `Esc` |
| त्वरित मिटाना (Brush उपयोग करते समय) | **दायाँ माउस बटन** दबाकर रखें |

## समर्थित फ़ॉर्मैट

- **छवियाँ:** JPG, PNG, WEBP, BMP, TIFF और अन्य सामान्य छवि फ़ॉर्मैट।
- **वीडियो:** MP4, MOV, MKV, AVI, WEBM, WMV, FLV… (Pro संस्करण)।

## समस्या निवारण

| लक्षण | क्या करें |
|---|---|
| खोलने पर Windows चेतावनी देता है | **More info → Run anyway** (अधिक जानकारी → फिर भी चलाएँ) पर क्लिक करें (ऐप सुरक्षित है, बस यह कोड-साइन्ड नहीं है)। |
| वीडियो नहीं खुलते | आप **Lite** संस्करण पर हैं — केवल छवियाँ। **Pro** संस्करण का उपयोग करें। |
| हटाने के बाद हल्के निशान रह जाते हैं | **अधिक पूरी तरह और किनारों से थोड़ा बाहर तक** पेंट करें, फिर दोबारा Run करें। |
| वीडियो बहुत धीमा है | सामान्य बात है — वीडियो भारी होता है; कृपया प्रतीक्षा करें, या ग्राफ़िक्स कार्ड वाली मशीन का उपयोग करें। |

## انسٹال اور کھولنا

1. انسٹالر چلائیں (یا اگر آپ کو کمپریسڈ آرکائیو ملا ہے تو فولڈر کو ان زپ کریں)۔
2. **AI Watermark Remover** کھولیں (ڈیسک ٹاپ سے، اسٹارٹ مینو سے، یا `.exe` فائل سے)۔

> **کیا Windows "Windows protected your PC" (ونڈوز نے آپ کے پی سی کی حفاظت کی) دکھا رہا ہے؟**
> یہ اُن ایپس کے لیے پہلے سے طے شدہ تنبیہ ہے جن کے پاس ادائیگی شدہ سائننگ سرٹیفکیٹ نہیں ہوتا — **یہ کوئی وائرس نہیں ہے**۔
> اسے کھولنے کے لیے **More info → Run anyway** (مزید معلومات ← بہرحال چلائیں) پر کلک کریں۔

پہلی بار کھولنے میں تھوڑا زیادہ وقت لگ سکتا ہے؛ بعد کے دفعات تیز ہوتی ہیں۔

**دو ایڈیشن ہیں:** **Pro** ایڈیشن (تصاویر + ویڈیو) اور **Lite** ایڈیشن (صرف تصاویر، ہلکا)۔
اِن میں فرق جاننے کے لیے ونڈو کا عنوان دیکھیں: **"Pro"** = مکمل ایڈیشن، **"Lite"** = ہلکا ایڈیشن (ویڈیوز نہیں کھول سکتا)۔

## استعمال کیسے کریں — 4 مراحل

### مرحلہ 1 — کوئی تصویر/ویڈیو کھولیں
- ایک یا زیادہ فائلیں چننے کے لیے **دستاویز** والے بٹن پر کلک کریں، یا پورا فولڈر کھولنے کے لیے **فولڈر** والے بٹن پر۔
- یا تصاویر/ویڈیوز کو سیدھا ونڈو میں **کھینچ کر چھوڑ دیں**۔
- کئی فائلوں کے ساتھ آپ کو نیچے تھمب نیل کی ایک پٹی ملتی ہے؛ ہر آئٹم پر ایک **×** ہوتا ہے جس سے اسے فہرست سے ہٹایا جا سکتا ہے۔

### مرحلہ 2 — جو ہٹانا ہے اس پر برش پھیریں
بائیں کالم میں کوئی ٹول چنیں، پھر واٹر مارک پر برش پھیریں/خاکہ بنائیں (برش پھیرا گیا حصہ **ہلکے سرخ** رنگ میں دکھائی دیتا ہے):

| ٹول | اس کا استعمال |
|----|---|
| **Brush** | واٹر مارک پر برش پھیریں (ماؤس کا بایاں بٹن دبائے رکھ کر گھسیٹیں)۔ برش کا سائز نیچے دیے گئے سلائیڈر سے ایڈجسٹ کریں۔ |
| **Box** | فوری طور پر ڈھانپنے کے لیے واٹر مارک پر ایک مستطیل کھینچیں۔ |
| **Eraser** | فالتو برش کیا ہوا حصہ مٹا دیں۔ |
| **Text** | کسی متن نما واٹر مارک کو بالکل ٹھیک ڈھانپنے کے لیے متن ٹائپ کریں۔ یہ متن صرف ایک **نشان** ہے — پروسیسنگ کے بعد یہ غائب ہو جاتا ہے اور تصویر میں شامل نہیں ہوتا۔ |

بہت زیادہ برش پھیر دیا؟ بائیں کالم میں **Undo / Redo / Clear all / Reset** استعمال کریں۔
ٹول کو غیر منتخب کرنے کے لیے دوبارہ اسی ٹول کے بٹن پر کلک کریں (یا **Esc** دبائیں)۔

> **نکتہ:** واٹر مارک کو پوری طرح ڈھانپیں اور تھوڑا سا باہر تک بھی پھیلائیں — کم برش کرنے کے مقابلے میں نتائج بہتر آتے ہیں۔

### مرحلہ 3 — Run پر کلک کریں
- **Run** — فی الوقت کھلی ہوئی تصویر/ویڈیو کو پروسیس کریں۔
- **Run all** — فہرست میں موجود **ایک ہی فریم سائز والی** ہر فائل کو بیچ میں پروسیس کریں
  (یہ صرف اُس وقت فعال ہوتا ہے جب 2 یا زیادہ فائلیں ہوں)۔

### مرحلہ 4 — محفوظ کریں
- **Save** پر کلک کریں۔ **ہر بار محفوظ کرنے پر، ایپ آپ سے آؤٹ پٹ فائلوں کے لیے کوئی فولڈر چننے کو کہتی ہے**۔
- اصل فائلیں رکھنا چاہتے ہیں؟ **"Add _clean"** آپشن کو فعال کریں — نئی فائلوں کے نام میں `_clean` کا سابقہ لگ جاتا ہے
  (مثلاً `photo.png` → `photo_clean.png`)، تاکہ اصل فائل پر دوبارہ لکھا نہ جائے۔

## ویڈیو پروسیسنگ (Pro ایڈیشن)

- ویڈیوز **Save بٹن سے محفوظ نہیں ہوتیں**۔ **Run** (ایک ویڈیو) یا **Run all**
  (کئی ویڈیوز) پر کلک کریں؛ شروع ہوتے وقت ایپ محفوظ کرنے کا فولڈر پوچھتی ہے۔
- ویڈیو کو AI کے ذریعے فریم بہ فریم پروسیس کیا جاتا ہے، اس لیے یہ **تصاویر کے مقابلے میں بہت سست ہوتا ہے** — بس اسے چلنے دیں۔

## دیکھنا اور دیگر سہولیات

- **ماؤس وہیل**: زوم اِن / آؤٹ · **درمیانی ماؤس بٹن دبائے رکھ کر گھسیٹنا**: پین کرنا · **F** یا **ڈبل کلک**: ونڈو کے مطابق فٹ کرنا۔
- **گھر** آئیکن: ہوم پیج کھولیں۔ **کرہ ارض** آئیکن: زبان تبدیل کریں (9 زبانیں معاون ہیں، بشمول ویتنامی)۔

## کی بورڈ شارٹ کٹس

| عمل | کی |
|---|---|
| Undo / Redo | `Ctrl+Z` / `Ctrl+Shift+Z` |
| برش کا سائز تبدیل کریں | `Ctrl` + ماؤس وہیل |
| ونڈو کے مطابق فٹ کریں | `F` یا ڈبل کلک |
| ٹول غیر منتخب کریں | `Esc` |
| فوری مٹانا (Brush استعمال کرتے وقت) | **ماؤس کا دایاں بٹن** دبائے رکھیں |

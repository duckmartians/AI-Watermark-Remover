Это приложение помогает **удалять логотипы, водяные знаки, подписи и текстовые надписи с изображений и видео**.
Просто **закрасьте то, что хотите убрать**, и приложение естественно заполнит это место. Всё работает
**на вашем собственном компьютере** — **интернет не нужен**, а ваши изображения и видео **никуда не загружаются**.

## Установка и запуск

1. Запустите установщик (или распакуйте папку, если вы получили сжатый архив).
2. Откройте **AI Watermark Remover** (с рабочего стола, из меню «Пуск» или через файл `.exe`).

> **Windows показывает «Windows protected your PC» (Windows защитил ваш компьютер)?**
> Это стандартное предупреждение для приложений без платного сертификата подписи — **это не вирус**.
> Нажмите **More info → Run anyway** (Подробнее → Выполнить в любом случае), чтобы открыть его.

Первый запуск может занять немного больше времени; последующие запуски будут быстрее.

**Существует две редакции:** редакция **Pro** (изображения + видео) и редакция **Lite** (только изображения, более лёгкая).
Чтобы различить их, посмотрите на заголовок окна: **«Pro»** = полная редакция, **«Lite»** = лёгкая редакция (не может открывать видео).

## Как пользоваться — 4 шага

### Шаг 1 — Откройте изображение/видео
- Нажмите кнопку **документ**, чтобы выбрать один или несколько файлов, или кнопку **папка**, чтобы открыть целую папку.
- Или **перетащите** изображения/видео прямо в окно.
- При нескольких файлах внизу появляется полоса миниатюр; у каждого элемента есть **×** для удаления из списка.

### Шаг 2 — Закрасьте то, что нужно удалить
Выберите инструмент в левом столбце, затем закрасьте/обведите водяной знак (закрашенная область отображается **мягким красным цветом**):

| Инструмент | Для чего использовать |
|----|---|
| **Кисть** | Закрашивайте водяной знак (удерживайте левую кнопку мыши и ведите). Регулируйте размер кисти ползунком ниже. |
| **Прямоугольник** | Растяните прямоугольник поверх водяного знака для быстрого закрытия. |
| **Ластик** | Сотрите лишнюю закраску. |
| **Текст** | Введите текст, чтобы точно закрыть водяной знак в виде текста. Этот текст — лишь **метка**: он исчезает после обработки и не впечатывается в изображение. |

Закрасили слишком много? Используйте **Отменить / Повторить / Очистить всё / Сбросить** в левом столбце.
Нажмите кнопку инструмента ещё раз (или клавишу **Esc**), чтобы отменить его выбор.

> **Совет:** закрывайте водяной знак полностью и заходите немного за края — результат получается лучше, чем при недостаточной закраске.

### Шаг 3 — Нажмите «Запустить»
- **Запустить** — обработать текущее открытое изображение/видео.
- **Запустить всё** — пакетно обработать все файлы **с одинаковым размером кадра** в списке
  (доступно только при наличии 2 или более файлов).

### Шаг 4 — Сохранить
- Нажмите **Сохранить**. **Каждый раз при сохранении приложение просит выбрать папку** для итоговых файлов.
- Хотите сохранить оригиналы? Включите параметр **«Добавить _clean»** — новые файлы получат суффикс `_clean`
  в имени (например, `photo.png` → `photo_clean.png`), так что оригинал не будет перезаписан.

## Обработка видео (редакция Pro)

- Видео **не сохраняются кнопкой «Сохранить»**. Нажмите **Запустить** (одно видео) или **Запустить всё**
  (несколько видео); приложение запросит папку для сохранения при старте.
- Видео обрабатывается кадр за кадром с помощью ИИ, поэтому это **гораздо медленнее, чем изображения** — просто дайте ему поработать.

## Просмотр и вспомогательные функции

- **Колесо мыши**: приблизить / отдалить · **Удерживайте среднюю кнопку мыши и ведите**: панорамирование · **F** или **двойной щелчок**: вписать в окно.
- Значок **домик**: открыть домашнюю страницу. Значок **глобус**: сменить язык (поддерживается 9 языков, включая вьетнамский).

## Горячие клавиши

| Действие | Клавиша |
|---|---|
| Отменить / Повторить | `Ctrl+Z` / `Ctrl+Shift+Z` |
| Изменить размер кисти | `Ctrl` + колесо мыши |
| Вписать в окно | `F` или двойной щелчок |
| Отменить выбор инструмента | `Esc` |
| Быстрое стирание (при работе кистью) | Удерживайте **правую кнопку мыши** |

---

## Поддерживаемые форматы

- **Изображения:** JPG, PNG, WEBP, BMP, TIFF и другие распространённые форматы изображений.
- **Видео:** MP4, MOV, MKV, AVI, WEBM, WMV, FLV… (редакция Pro).

## Устранение неполадок

| Проблема | Что делать |
|---|---|
| Windows предупреждает при открытии | Нажмите **More info → Run anyway** (Подробнее → Выполнить в любом случае) (приложение безопасно, оно просто не подписано сертификатом). |
| Не открываются видео | У вас редакция **Lite** — только изображения. Используйте редакцию **Pro**. |
| После удаления остаются слабые следы | Закрасьте **более полно и слегка за края**, затем запустите снова. |
| Видео обрабатывается очень медленно | Это нормально — видео тяжёлое; подождите или используйте компьютер с видеокартой. |
